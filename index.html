Visão Geral do Projeto

Entenda a proposta e o valor da arquitetura.

O Problema

Aplicações que executam análises de dados pesadas diretamente em requisições HTTP sofrem com lentidão, timeouts e baixa escalabilidade. A API fica bloqueada, prejudicando a experiência do usuário e a capacidade de resposta do sistema como um todo.

A Solução

O Async Data Analysis Service desacopla a recepção da requisição do seu processamento. A API responde instantaneamente com um ID de trabalho, enquanto a análise é enfileirada e executada em segundo plano por workers dedicados, garantindo alta performance e resiliência.

Arquitetura do Sistema

Uma visão clara de como os componentes se comunicam.

1. API (Gateway)

Recebe requisições, autentica e publica jobs na fila.

2. RabbitMQ

Message Broker que gerencia as filas de tarefas de forma confiável.

3. Worker

Consome mensagens da fila e executa a análise dos dados.

4. MongoDB

Armazena os dados brutos e os resultados processados.

Funcionalidades Principais

Os pilares que tornam este projeto robusto e moderno.

Processamento Assíncrono

Respostas instantâneas da API, com tarefas pesadas executadas em background.

Segurança Robusta

Autenticação com JWT e OAuth2, garantindo acesso seguro aos endpoints.

Escalabilidade Horizontal

Adicione mais workers conforme a demanda cresce, sem impactar a API.

Resiliência e Confiabilidade

Uso de Dead Letter Queues (DLQ) para tratar falhas no processamento.

Ambiente Containerizado

Docker e Docker Compose para consistência entre desenvolvimento e produção.

CI/CD Automatizado

Pipeline com GitHub Actions para automação de testes e deploy contínuo.

Fluxo de Uso da API

Três passos simples para iniciar uma análise.

1

Autenticação

Obtenha um token de acesso via `/api/auth/login`.

2

Solicitar Análise

Envie a requisição para `/api/analysis/...` e receba um `job_id`.

3

Consultar Resultado

Verifique o status e o resultado em `/api/analysis/results/{job_id}`.

Tecnologias Utilizadas

As melhores ferramentas para um serviço moderno e eficiente.

Python & FastAPI

Base da nossa API, escolhido pela alta performance assíncrona (ASGI), tipagem de dados com Pydantic que garante a validação e a geração automática de documentação interativa, acelerando o desenvolvimento e a integração.

RabbitMQ

Atua como o sistema nervoso da arquitetura, desacoplando a API dos workers. Garante que nenhuma solicitação seja perdida, gerenciando filas de tarefas de forma persistente e distribuindo a carga de trabalho de maneira eficiente.

MongoDB

Como banco de dados NoSQL, oferece a flexibilidade necessária para armazenar tanto os dados brutos, com esquemas variados, quanto os resultados estruturados das análises. Sua escalabilidade horizontal acompanha o crescimento do volume de dados.

Docker & Docker Compose

Todo o ambiente é containerizado, eliminando problemas de compatibilidade entre desenvolvimento, teste e produção. O Docker Compose orquestra todos os serviços, simplificando o setup e garantindo a reprodutibilidade da infraestrutura.

Pytest

Garante a qualidade e a confiabilidade do código através de uma suíte de testes automatizados. A cobertura de testes unitários e de integração previne regressões e assegura que novas funcionalidades não quebrem o sistema existente.

GitHub Actions

Automatiza o ciclo de vida do desenvolvimento. A cada nova versão, o pipeline de CI/CD é acionado para rodar os testes, construir as imagens Docker e realizar o deploy em produção, garantindo entregas rápidas e seguras.

Implementação Simplificada

A infraestrutura do projeto é provisionada de forma automatizada com scripts de build e orquestração de containers, garantindo um ambiente consistente e pronto para uso.

1. Configuração de Ambiente

Scripts automatizados preparam todo o ambiente, configurando variáveis, segredos e dependências.

2. Orquestração de Containers

O Docker Compose inicia e gerencia todos os serviços (API, Worker, DB) de forma integrada e isolada.

3. Pronto para Uso

Com os serviços em execução, a API está disponível para receber requisições e processar análises de dados.
